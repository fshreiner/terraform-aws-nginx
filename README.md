# Terraform AWS – EC2 com Nginx em VPC customizada

Projeto de **Infraestrutura como Código (IaC)** utilizando **Terraform** para provisionar uma infraestrutura na **AWS**, com foco em **boas práticas** e **uso em DevOps/Cloud**.

O projeto cria uma **VPC**, uma **subnet privada**, configura os **Security Groups** e provisiona uma **instância EC2 Ubuntu** com **Nginx instalado automaticamente** que pode ser acessada somente via **Load Balancer** criado automaticamente.
